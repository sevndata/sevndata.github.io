使用IntelliJ IDEA生成代码

2019-05-26 IntelliJ IDEA ， POJO

IntelliJ IDEA可以根据数据库生成“POJO”代码，在此基础上，可以通过修改Generate POJOs.groovy文件来生成自定义的代码如：Controller, Service。

2. 生成代码

  3. 点击Scripted Extensions -> Generate POJOs.groovy
